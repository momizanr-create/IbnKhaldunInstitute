📦 Snapshot ও Download

Static snapshot তৈরি করুন — index.html-এর জন্য

কী এই সিস্টেম?

🔥 ওয়েবসাইটের স্পিড বাড়ানোর সিস্টেম: এই Snapshot সিস্টেম admin panel থেকে সব data — কোর্স, ইন্সট্রাক্টর, রিভিউ, FAQ, settings — একটি single JSON file (data-snapshot.json) এ একসাথে save করে রাখে।

✅ ফলাফল: index.html যখন user প্রবেশ করে, সব data 1টি fetch এ পেয়ে যায় — আগের মত প্রতিটা সেকশনের জন্য আলাদা API call করে না। সাইট দ্রুত লোড হয়।

🔄 Admin panel থেকে যেকোনো পরিবর্তন save করলে snapshot স্বয়ংক্রিয়ভাবে rebuild হয়। আপনি নিচের বাটন দিয়ে manual rebuild করতে পারেন বা snapshot ডাউনলোড করতে পারেন।
